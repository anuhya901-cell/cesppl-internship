# Week 9 – CESPPL Dataset Inventory & Preparation

## Overview

This week focused on preparing and organizing the CESPPL dataset for the image classification project. The objective was to analyze the dataset, generate metadata, and create a clean train/validation/test split for model training.

## Tasks Completed

- Performed complete dataset inventory and analysis.
- Counted images for each class.
- Generated file size and extension summaries.
- Identified and documented excluded or invalid files.
- Created structured dataset metadata files.
- Split the dataset into training, validation, and testing sets.
- Verified dataset consistency before training.

## Files Included

- `w9_cesppl_inventory.ipynb` – Dataset inventory and analysis notebook.
- `inventory.csv` – Complete image inventory.
- `extension_counts.csv` – Distribution of file extensions.
- `file_size_summary.csv` – Dataset file size statistics.
- `excluded_files.csv` – List of excluded files.
- `CLASSES.md` – Class names used for classification.
- `train.csv` – Training dataset split.
- `val.csv` – Validation dataset split.
- `test.csv` – Testing dataset split.

## Outcome

- Prepared a clean and organized dataset for deep learning.
- Generated all metadata required for reproducible experiments.
- Successfully completed the dataset preparation phase, enabling model development in Week 10.
