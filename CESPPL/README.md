# CESPPL Dataset Inventory (Week 9)

This folder contains the complete inventory and exploratory analysis of the CESPPL operational image dataset received from the mentor.

## Contents

- `w9_cesppl_inventory.ipynb` – Dataset inventory notebook
- `CLASSES.md` – Canonical class names
- `inventory.csv` – Complete inventory summary
- `excluded_files.csv` – Excluded or invalid files (if any)
- `extension_counts.csv` – File extension statistics
- `file_size_summary.csv` – File size statistics

## Dataset

- Number of classes: 10
- Total valid images: 4179
- Largest class: MANUAL BEACH CLEANING
- Smallest class: BIN LIFTING

This inventory was created before any preprocessing or model training.
